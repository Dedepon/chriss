Voici les différentes étapes pour l'utilisation du site.
  • S'enregistrer :
    • Cette étape est nécessaire afin de nous permettre de suivre vos cadeaux et pouvoir vous remercier.
    1. Cliquez sur le bouton "S'enregistrer"
    2. Entrez un nom d'utilisateur (login)
    3. Entrez et confirmez un mot de passe
    4. Cliquez sur "Continuer"
    Si vous revenez plus tard, ou sur un autre ordinateur/smartphone, vous pourrez vous authentifier.
    1. Cliquez sur le bouton "S'authentifier"
    2. Entrez un nom d'utilisateur (login)
    3. Entrez votre mot de passe
    4. Cliquez sur "Continuer"
  • Choisir les cadeaux :
    • Une fois enregistré et authentifié, vous êtes sur la liste des cadeaux.
    1. Cliquez sur le cadeau que vous voulez offrir
    2. La liste des informations du cadeau : commentaire, prix, nombre offert et un lien vers le site du revendeur.
    3. Choisissez la manière de nous faire parvenir ce cadeau : livrer chez nous ou faire un virement pour que l'on puisse le prendre nous-même
    4. Choisissez le nombre de fois que vous voulez nous offrir ce cadeau
    5. Cliquez sur "Continuer"
    Si aucun cadeau ne vous tente, il reste la "tirelire", qui vous permet de participer financièrement à notre vie de tous les jours et les dépenses quotidiennes.
  • Voir le récapitulatif des cadeaux choisis (panier) :
    • Le panier vous permet de voir le récapitulatif des cadeaux que vous avez choisis.
    1. Cliquez sur le panier, en haut à droite de l'écran
    2. Vous pouvez modifier le nombre de cadeau en changeant la quantité et en cliquant sur "Mettre à jour"
    3. Vous pouvez supprimer un cadeau en cliquant sur le panier barré
    4. Le total est séparé en fonction des manières de nous faire parvenir les cadeaux que vous avez choisies
    5. Lorsque vous êtes satisfait des cadeaux, cliquez sur "Confirmer"
    6. Nos informations de notre compte bancaire et notre adresse postale sont données sur cet écran
    7. Un rappel des cadeaux que vous avez déjà offert sont rappelés en bas de l'écran
    Lorsque vous êtes satisfait, cliquez sur "confirmer" pour nous informez que vous comptez nous offrir les cadeaus que vous avez choisi.
  • Entrer vos coordonnées :
    • Afin de pouvoir vous remercier pour vos cadeaux, vous pouvez remplir vos coordonnées, et laisser un petit mot.
    1. Cliquez sur l'icône du profil
    2. Rentrez vos coordonnées
    3. Laissez-nous un petit message
    4. Cliquez sur "Sauvegarder"
    Cette étape n'est pas obligatoire. Vous êtes libres de ne pas la faire si vous ne souhaitez pas communiquer vos données personnelles.
